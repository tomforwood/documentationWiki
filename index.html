DocumentationWiki

This project creates a user editable webapp that allows the users of a undocumented, closed source API to collaboratively document its public features. It was created to allow modders of Kerbal Space Program to document it's extensive API.

The code comprises 3 executables in a mixture of Java and C# because I'm a java programmer really

Reflector

C# — This module uses c# reflection in order to extract the details of every class in the KSP API

initialPopulate

Java — This module builds upon the work already done by The_Duck with Previous KSP Docs. It uses an antlr c# grammar to parse the documented c# files from that project and extract the documentation comments.

documentationWiki

Java, AngularJS — This creates a dropwizard server. The server queries for information about a requested class, combines the reflected API information with the most recent comments from the initial populate and from the users. The angular javascript is then rsponsible3 for formatting the data into html.

Contributing

To Work on this project you will need

  1. A Java development environment with maven
  2. A C# development environment
  3. A local Mongo database

To run the app locally

  • Fork and clone the repository as per GitHub normal
  • Run Reflector with no args to populate the reflected data collection
  • Optionaly to create an initial documentation population
    • Download the initial source
    • Run InitialPopluate to populate the annotated collection
  • Start DocumentationWikiApplication
  • Go to http://localhost:8080/ to view the app

Make any changes, commit, push to your fork, and then create a pull request for your changes to be incorporated
