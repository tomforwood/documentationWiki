Contributing

Anyone can edit this documentation. To edit the comment attached to an element simply click on the comment box and make your edits

You can use basic html in the comments (it will be sanitized by angular's sanitizer). The editor also supports most of the standard microsoft documentation tags

Links are supported using both with <see cref= (for class links) and with HTML <a> links. To link to a method within a class use a link of the form <a href="#/classes/Part?scrollTo=ActivatesEvenIfDisconnectedF"> where Part is the name of the class, ActivatesEvenIfDisconnected is the name of the member in that class to show and F is one of F, P, M, C, N indicating of the linked member is a field, property, method, constructor or an nested inner type.

Once you are finished editing PRESS SAVE at the bottom of the page to save your changes.

The structure of the documentation is created by reflection on the KSP assembly. The comments come from users with an initial population from the comments in the project created by The_Duck/Anatid.
