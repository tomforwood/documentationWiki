Contributing

Anyone can edit this documentation. To edit the comment attached to an element simply click on the comment box and make your edits

You can use basic html in the comments (it will be sanitized by angular's sanitizer)

Then PRESS SAVE at the bottom of the page to save them.

The structure of the documentation is created by reflection on the KSP assembly. The comments come from users with an initial population from the comments in the project created by The_Duck/Anatid.
